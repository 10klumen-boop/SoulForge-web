# Контент в JSON

Правь таблицы здесь, не в огромных `.js`:

| Файл | Глобалы |
|------|---------|
| `story-zones.json` | `STORY_ARC`, `RACE_*`, `STORY_BEATS`, `FARM_ZONES`, `ZONE_RACE_BONUS` |
| `quest-content.json` | `QUESTS_PER_ZONE`, `ZONE_BOSSES`, `QUEST_NPC_BY_RACE_ZONE`, … |
| `achievements.json` | title/desc/reward/иконки (`ACHIEVEMENTS_META`, …); `test()` в `achievement-data.js` |
| `zone-chapter-rewards.json` | `ZONE_CHAPTER_REWARDS` |
| `passive-skills.json` | `PASSIVE_SKILLS`, `RACE_PASSIVE_SKILL_IDS`, `CLASS_PASSIVE_SKILL_IDS` |
