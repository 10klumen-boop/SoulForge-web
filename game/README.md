# SoulForge Lineage 2 — Enchant Sim

Фан-симулятор заточки в духе классического Lineage 2. Один `index.html` + модули в `src/`.

## Как запустить

Открой `index.html` в браузере или запусти десктоп через `../start.bat` (Tauri dev).

## Основное

- **Инвентарь** — оружие, кристаллы, drag-and-drop, лимит ячеек
- **Заточка +0…+16** — безопасная зона, три типа свитков по грейдам D/C/B/A
- **Экономика adena** — свитки, продажа с +4, шахта, мастерская, ачивки
- **Шахта** — кликер с гномами, руда, редкий лут
- **Мастерская** — заряды / соски
- **Аксессуары**, FunPay-сделка, достижения, журнал
- **Сохранения** — localStorage / desktop `.sf2`

## Управление

| Клавиша | Действие |
|---------|----------|
| `Space` | Заточить |
| `S` | Продать (с +4) |
| `N` | В инвентарь |
| `Esc` | Назад |
| `M` | Звук вкл/выкл |

Dev-панель: `Ctrl+Shift+D` (только в dev-сборке).
