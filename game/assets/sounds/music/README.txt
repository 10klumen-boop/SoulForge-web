SoulForge — музыка главного меню
================================

menu_theme.m4a — «The Call of Destiny» (Lineage II: Chaotic Chronicle, NCsoft).
Тот же эпичный мотив, что на экране входа в оригинальной L2.

Перекачать / сменить трек:
  pip install yt-dlp
  python game/tools/fetch_l2_music.py
  python game/tools/fetch_l2_music.py --track interlude

После замены файла — поднять SOUND_VER в game/src/03-audio.js и Ctrl+F5.

Файл не в git? Запустите fetch_l2_music.py один раз.
