SoulForge — звуки из L2 Interlude (NevesOma_il_sounds)
======================================================

Полный пак не в git. В репозитории — отобранные wav в game/assets/sounds/.

Пересборка:
  python game/tools/pick_l2_sounds.py
  python game/tools/pick_l2_sounds.py "D:/path/NevesOma_il_sounds"

Скрипт: trim silence, normalize peak, trim длинных, fade для loop ambient.
